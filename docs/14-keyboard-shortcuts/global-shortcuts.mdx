# Global Shortcuts

Global shortcuts work regardless of which tab or panel is currently focused. They are registered at the application level and cannot be blocked by terminal input or editor focus.

## Full Reference

| Action | Shortcut |
|---|---|
| New Tab | `Cmd+T` |
| Close Tab | `Cmd+W` |
| Command Palette | `Cmd+K` or `Cmd+Shift+P` |
| Shortcuts Reference | `Cmd+?` |
| Settings | `Cmd+,` |
| Toggle Sidebar | `Cmd+B` |
| Toggle AI Panel | `Cmd+Shift+A` |
| Toggle Full Screen | `Cmd+Ctrl+F` |
| Next Tab | `Cmd+]` or `Cmd+Shift+]` |
| Previous Tab | `Cmd+[` or `Cmd+Shift+[` |
| Switch to Tab 1 | `Cmd+1` |
| Switch to Tab 2 | `Cmd+2` |
| Switch to Tab 3 | `Cmd+3` |
| Switch to Tab 4 | `Cmd+4` |
| Switch to Tab 5 | `Cmd+5` |
| Switch to Tab 6 | `Cmd+6` |
| Switch to Tab 7 | `Cmd+7` |
| Switch to Tab 8 | `Cmd+8` |
| Switch to Tab 9 | `Cmd+9` |
| Reload Window | `Cmd+R` (when no terminal is focused) |
| Quit | `Cmd+Q` |

## Tab Switching by Number

`Cmd+1` through `Cmd+9` switch to the tab at that position in the tab bar. `Cmd+9` always jumps to the last tab regardless of total count, consistent with browser behavior.

## Reload Window

`Cmd+R` reloads the Tauri WebView. This is safe to use when no terminal tab is focused — it restores the UI without affecting running processes that are managed by the Rust backend. If a terminal tab is focused, `Cmd+R` is intercepted by the terminal and sent as input rather than triggering a reload.

:::tip
Use `Cmd+?` to open the interactive Shortcuts Reference dialog at any time. It displays all shortcuts organized by category and allows you to customize bindings.
:::

:::tip
`Cmd+B` is the fastest way to reclaim horizontal screen space when working in terminals or SFTP — toggle the sidebar off when you do not need the file explorer.
:::
