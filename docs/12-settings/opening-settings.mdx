# Opening Settings

Labonair's Settings window gives you a single place to configure every aspect of the application — from fonts and themes to AI model keys and keyboard shortcuts.

## Opening the Settings Window

**Keyboard shortcut:** `Cmd+,`

This is the fastest way to open Settings from anywhere in the app.

**Via Command Palette:**

1. Press `Cmd+K` to open the Command Palette.
2. Type `Open Settings` and press `Enter`.

## Window Layout

The Settings window is a dedicated macOS window — it opens separately from the main workspace and does not replace any open tabs. You can keep it open alongside your terminal sessions.

The window uses a **sidebar-detail layout**:

- **Left sidebar** — contains a search bar at the top and a list of settings categories below it.
- **Right content area** — displays the settings for whichever category is selected in the sidebar.

## Settings Categories

| Category | What it controls |
|---|---|
| General | App font, startup behavior, language |
| Appearance | Theme picker, UI font, background image |
| Terminal | Font, cursor, scrollback, line height |
| Editor | Auto-save, word wrap, tab size, line numbers |
| File Manager | Default view, SFTP conflict behavior, hidden files |
| Models | AI API keys, default model, custom provider endpoint |
| Agents | Max agent steps, auto-execute tool toggles |
| Keyboard Shortcuts | View and remap all keyboard shortcuts |
| About | Version info, update check, licenses |

## Global Search

Type in the search bar at the top of the sidebar to filter settings across **all categories simultaneously**. The content area updates in real time as you type, showing only matching settings regardless of which category they belong to. This is the fastest way to find a specific setting without knowing which section it lives in.

## Instant Apply

Settings in Labonair take effect immediately when you change them — there is no Save button and no confirmation step. Close the Settings window at any time; your changes are already persisted.
